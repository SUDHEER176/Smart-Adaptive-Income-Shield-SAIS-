# SAIS - Smart Adaptive Income Shield

AI-Powered Parametric Income Protection for India's Gig Economy.

## 🚀 Vision
SAIS is designed to bridge the income gap for gig workers (delivery partners, ride-hailing drivers, etc.) caused by external disruptions like extreme weather, high pollution, or sudden demand drops.

---

## ✨ Key Innovations

1.  **WhatsApp-First Architecture**: No app to download. Workers interact entirely via WhatsApp—a platform they already use daily.
2.  **Parametric Triggers**: Claims are not based on subjective reports but on verified external data (AQI, Weather, Traffic).
3.  **Real-Time Decision Engine**: Automated claim verification using a 0-100 Confidence Score.
4.  **Instant Payouts**: Once a trigger is verified and downtime is confirmed, payouts are initiated directly to UPI.

---

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, Lucide React, Recharts (Responsive Dashboard)
- **Backend API**: Node.js, Express
- **Database / Session Management**: Supabase (PostgreSQL)
- **Messaging**: Twilio WhatsApp API
- **Verification Logic**: Multi-signal validation engine

---

## 🧠 Application Workflow

1.  **Registration**: Worker sends `REGISTER` on WhatsApp.
2.  **Monitoring**: System monitors active triggers (e.g., Heavy Rain in Mumbai).
3.  **Trigger Detection**: If a trigger is active, workers in that zone are notified.
4.  **Verification**: System checks worker activity, peer reports, and external signals.
5.  **Payout**: Automated credit to UPI for verified income loss.
